GitHub Action Runner 架構

基於 Docker-in-Docker 的動態臨時自託管 Runner 平台

安全性

  • • GitHub App 驗證（JWT，非 PAT）
  • • HMAC-SHA256 Webhook 簽章驗證
  • • 臨時 Runner（任務完成後自動銷毀）
  • • TLS 憑證透過 Docker Volume 共享
  • • 隔離的 Docker 網路環境

基礎設施

  • • Docker-in-Docker 容器編排
  • • Cloudflare Tunnel 實現 NAT 穿透
  • • 根據任務標籤智慧選擇映像檔
  • • 每日排程自動拉取最新 Runner 映像檔
  • • 可自訂 Runner 映像 + 基礎 Ubuntu Runner
